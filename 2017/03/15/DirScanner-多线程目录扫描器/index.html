  • 程序使用

    -h够看到如下帮助信息
Usage: [-u <Website/Target_url>][-t <Scan thread num>]

Options:
  -h, --help     查看帮助信息
  -u 必选,加网站链接  Example:-u http://www.example.com
  -s 必选,网站开发脚本   Example:-s asp
  -t 可选,默认线程数为10   Example:-t 50
